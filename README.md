Notes API:
<br>
A production-ready Notes Management API built using Django REST Framework.

#Features: 
  <br>
  User Registration
  <br>
  JWT Authentication
  <br>
  Create Notes
  <br>
  Read Notes
  <br>
  Update Notes
  <br>
  Delete Notes
  <br>
  Pagination
  <br>
  Filtering & Search
  <br>
  PostgreSQL Database
  <br>
  Interactive Swagger Documentation
  <br>
  Deployed on Render

#Installation:
  <br>

  Clone repository:
  <br>
  git clone <repository-url>
  
  Move into project folder:
  <br>
  cd notes_api
  
  Install dependencies:
  <br>
  pip install -r requirements.txt
  
  Run migrations:
  <br>
  python manage.py migrate
  
  Start server:
  <br>
  python manage.py runserver
