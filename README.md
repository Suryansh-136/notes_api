Notes API:
A production-ready Notes Management API built using Django REST Framework.

#Features: 
  <br>
  User Registration
  <br>
  JWT Authentication
  <br>
  Create Notes
  <br>
  Read Notes
  <br>
  Update Notes
  Delete Notes
  Pagination
  Filtering & Search
  PostgreSQL Database
  Interactive Swagger Documentation
  Deployed on Render

#Tech Stack:
  Python
  Django
  Django REST Framework
  PostgreSQL
  JWT Authentication
  Swagger (drf-yasg)
  Git & GitHub
  Render

#Installation:

  Clone repository:
  git clone <repository-url>
  
  Move into project folder:
  cd notes_api
  
  Install dependencies:
  pip install -r requirements.txt
  
  Run migrations:
  python manage.py migrate
  
  Start server:
  python manage.py runserver
